# DozingCat 网站安全保护说明

## 🔒 安全保护功能

本网站采用多层次安全保护机制，确保用户数据和隐私安全。

### 1. 数据加密

#### 端到端加密
- **数据传输加密** - 所有敏感数据在传输过程中使用加密算法保护
- **数据存储加密** - 本地存储的数据经过加密处理
- **密钥管理** - 使用安全密钥进行加密/解密操作

#### 加密算法
```javascript
- XOR加密算法
- Base64编码
- 安全密钥保护
- 随机盐值生成
```

### 2. 输入验证与清理

#### 输入清理
- **XSS防护** - 移除潜在的跨站脚本攻击代码
- **SQL注入防护** - 清理特殊字符防止SQL注入
- **恶意代码过滤** - 过滤JavaScript、data:等危险协议

#### 数据验证
- **邮箱验证** - 使用正则表达式验证邮箱格式
- **手机号验证** - 验证手机号格式（中国大陆）
- **密码强度检查** - 检查密码复杂度

### 3. 密码安全

#### 密码强度评估
- **弱密码** - 长度不足8位或缺少大小写字母和数字
- **中等密码** - 包含大小写字母和数字
- **强密码** - 包含大小写字母、数字和特殊字符

#### 密码哈希
- **单向哈希** - 密码经过哈希处理，不可逆
- **盐值加密** - 每个密码使用随机盐值
- **安全存储** - 哈希后的密码安全存储

### 4. 敏感数据保护

#### 数据脱敏
- **手机号脱敏** - 138****8888
- **邮箱脱敏** - ab***@example.com
- **银行卡号脱敏** - ****1234
- **银行账户脱敏** - 1234********5678

#### 安全存储
- **加密存储** - 敏感数据加密后存储
- **访问控制** - 限制对敏感数据的访问
- **日志记录** - 记录所有安全相关事件

### 5. 防护措施

#### 客户端防护
- **禁止右键** - 防止查看源代码
- **禁止F12** - 防止打开开发者工具
- **禁止Ctrl+U** - 防止查看页面源代码
- **禁止Ctrl+Shift+I/J/C** - 防止打开开发者工具

#### 安全头部
- **X-Content-Type-Options** - 防止MIME类型嗅探
- **X-Frame-Options** - 防止点击劫持
- **X-XSS-Protection** - 启用XSS过滤器
- **Content-Security-Policy** - 内容安全策略

### 6. 安全日志

#### 事件记录
- **支付事件** - 记录所有支付操作
- **登录事件** - 记录用户登录/登出
- **数据访问** - 记录敏感数据访问
- **异常事件** - 记录安全相关异常

#### 日志管理
- **自动清理** - 保留最近100条记录
- **时间戳** - 每条记录包含精确时间
- **用户代理** - 记录访问者信息

### 7. 安全徽章

#### 视觉提示
- **安全图标** - 🔒 锁图标显示安全状态
- **动画效果** - 脉冲动画提醒用户
- **自动显示** - 页面加载时自动显示
- **自动隐藏** - 5秒后自动隐藏

#### 安全信息
- **保护状态** - "安全保护已启用"
- **加密状态** - "所有数据已加密传输"

## 🔐 使用说明

### 开发者集成

```javascript
// 数据加密
const encrypted = SecurityUtils.encrypt(data);

// 数据解密
const decrypted = SecurityUtils.decrypt(encrypted);

// 输入清理
const cleanInput = SecurityUtils.sanitizeInput(userInput);

// 密码验证
const passwordCheck = SecurityUtils.validatePassword(password);

// 敏感数据脱敏
const maskedPhone = SecurityUtils.maskSensitiveData(phone, 'phone');

// 安全存储
SecurityUtils.secureLocalStorage.setItem(key, value);
const value = SecurityUtils.secureLocalStorage.getItem(key);

// 安全日志
SecurityUtils.logSecurityEvent('event_type', details);
```

### 安全检查

```javascript
// 检查安全状态
const securityStatus = SecurityUtils.checkSecurity();
console.log(securityStatus);
// { https: true, cookiesEnabled: true, doNotTrack: false, secureContext: true }
```

## ⚠️ 安全提示

1. **使用强密码** - 包含大小写字母、数字和特殊字符
2. **定期更换密码** - 建议每3个月更换一次
